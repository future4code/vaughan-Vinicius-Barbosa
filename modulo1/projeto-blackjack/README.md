<h1>Projeto black jack</h1>

```
Projeto e um jogo de cartas onde o objetivo é derrotar o dealer(Computador). Para fazer isso, você deve ter uma mão em que a pontuação seja mais elevada do que a mão do dealer(Computador), mas não exceda 21 pontos no valor total. Como alternativa, você pode ganhar tendo uma pontuação menor que 22 quando o valor da mão do dealer ultrapassar 21.
```
